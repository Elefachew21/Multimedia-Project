Project Overview

This project is a multimedia-based tourism promotion documentary developed to showcase selected Ethiopian historical, cultural, and natural heritage sites. The project uses multimedia elements such as images, videos, text, background music, transitions, and visual effects to create an engaging tourism promotion video.

The documentary highlights the rich history, culture, and natural beauty of Ethiopia through professionally edited multimedia content.

---

Featured Tourist Destinations

1. Lalibela Rock-Hewn Churches
2. Axum Tsion Mariam Church
3. Axum Obelisks
4. Fasil Ghebbi (Gondar Castles)
5. Sof Omar Cave

---

- Increase awareness of historical and cultural heritage.
